### Major Changes
- Migrated to AppKit (Reown) for wallet connections
- Updated all currency displays to STT format
- Improved wallet connection UX

### Improvements
- Simplified wallet state management
- Better error handling
- Enhanced documentation

### Breaking Changes
- Wallet connection API changed to AppKit hooks
- Currency format changed from USD to STT

See APPKIT_MIGRATION.md for migration details.
