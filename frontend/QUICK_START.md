# Quick Start Guide

## For Users

1. **Install Wallet**
   - Install MetaMask or compatible wallet
   - Add Somnia Testnet network

2. **Get STT Tokens**
   - Use Somnia faucet
   - Request test tokens

3. **Connect Wallet**
   - Click "Connect Wallet" button
   - Approve connection in wallet

4. **Start Earning**
   - Go to Deposit page
   - Enter deposit amount
   - Confirm transaction

## For Developers

1. **Clone Repository**
   ```bash
   git clone <repo-url>
   cd yieldmind/frontend
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Start Developing**
   - Open http://localhost:3000
   - Make changes
   - See updates live
