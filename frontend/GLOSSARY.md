# Glossary

## Terms

### AppKit
Modern wallet connection library (formerly Web3Modal) by Reown.

### STT
Somnia Test Token - the native token of Somnia Testnet.

### Wagmi
React Hooks library for Ethereum, used with AppKit.

### YieldMind
AI-powered DeFi investment platform on Somnia.

### Vault
Smart contract that holds user deposits and manages investments.

### APY
Annual Percentage Yield - the return rate on investments.

### TVL
Total Value Locked - total amount of assets in the platform.

### Reown
Company behind AppKit (formerly WalletConnect).

## Acronyms

- **DeFi**: Decentralized Finance
- **UI**: User Interface
- **UX**: User Experience
- **API**: Application Programming Interface
- **RPC**: Remote Procedure Call
