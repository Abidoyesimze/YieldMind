# Troubleshooting Guide

## Wallet Connection Issues

### Wallet Not Connecting
1. Check if wallet extension is installed
2. Ensure wallet is unlocked
3. Try refreshing the page
4. Check browser console for errors

### Wrong Network
1. Click "Switch Network" button
2. Manually switch in wallet extension
3. Ensure Somnia Testnet is added to wallet

### Transaction Failures
1. Check if you have sufficient STT balance
2. Verify contract addresses are correct
3. Check network congestion
4. Review transaction error messages

## Common Errors

### "MetaMask not detected"
- Install MetaMask extension
- Refresh the page
- Check if extension is enabled

### "Please switch to Somnia Testnet"
- Add Somnia Testnet to your wallet
- Use network switch button
- Verify chain ID is 50312

### "Insufficient funds"
- Check your STT balance
- Ensure you have enough for gas fees
- Verify transaction amount

## Getting Help

1. Check documentation
2. Review error messages
3. Open an issue on GitHub
4. Contact support
